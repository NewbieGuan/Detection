1.Detection项目用于拍摄图片，得到多帧8192×2048的图片，并截取每张图片中间2048×2048的图片拼接起来得到一张大图，这张大图用BatchTest进行处理得到红黑图与excel。

2.此项目需配置opencv、MPC2810、Sapera LT三个库
MPC2810库下载：www.leetro.com/down/MPC2810A%20V1.0.17-32bit%20Win7.rar
Sapera LT库下载：sapera_lt_831_sdk.exe 或者在http://51camera.com.cn/show-25-12-1.html的“软件及例程下载”中下载，这个网址就是此项目相机的选型。

已经在项目中属性管理器的Debug|x64和Release|x64配置了这几个库的属性表，使用时请以按照自己电脑的环境修改。
2.1修改方法一：可右击属性页->属性，修改包含目录、库目录和附加依赖项这几个选项。
2.2修改方法二：将属性页中用到的这三个库的.h文件 .lib文件 .dll文件全部放到源程序文件夹。

3.关于项目的一些参数
电机转1圈前进10mm（这个项目做了两台机器，这里指第二台机器，第一台机器与第二台电机型号不同，其他都相同）
电机接受1000个脉冲前进1mm
相机接受4个脉冲取1行
所以相机前进1mm扫描250行，所以在移动方向一个像素就是4μm。

那么原图就是
宽8192像素，实际44mm
宽一个像素5.371μm
高一个像素为4μm

应设置扫描行数为2048*5.371/4=2750，再进行压缩，压缩至2048，这样长宽比例一致。

但我们实际设置一个像素为5.378μm（因为第一台机器是这么定的，而支架与相机等都没变，所以直接沿用了）

得到的一帧图像为8k*2k
